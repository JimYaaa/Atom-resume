< h2 > 自我介紹 < / h2 >

< p > Stay Hungry, Stay Foolish. < / p >

< p > 我是葉儒敬 (Jim)，今年26歲，來自台南，現居台南。 < / p >

< p >

2012年畢業於高苑科技大學電子系航空電子組後，繼續待在學生時期打工的Pizza店上班，在因緣際會之下偶然接觸到前端這方面的資訊，心想或許可以試試看，就這樣開始了自學前端技術的日子，一開始遇到很多的困難跟挫折，但我沒有放棄，並且在做中學的過程中一點一點的慢慢累積信心與興趣，也在去年開始決定轉換跑道，今年三月也順利應徵到『作好數位』前端工程師一職，但也因為公司需要綁約兩年加上在公司裡比較沒辦法接觸到Javascript的實務練習，所以選擇離職。

< / p >

< p >

雖然在『作好數位』只待了短短的三個月，但這三個月的經歷卻讓我進步不少，例如瀏覽器兼容性、排版技巧、更簡潔的Css寫法、寫出更符合Html語意Code、以及在規劃版面時所需要考量到的細節等等，在任職的這段期間大致上都是以Html、Css為主在做一些網頁切版的工作，也因為有這三個月的工作經歷讓我對前端這個行業更有興趣也更有信心。

< / p >

< p >

在作好數位工作期間，開始付費參加了一些前端網頁開發的課程，也透過許多網路上的資源來學習，發現自己對於這一塊很有興趣，不斷地想去瞭解更多，也決定在在未來轉換跑道至這領域。 雖然現在的我還有很多需要學習的地方，為了期許自己在未來可以成為獨當一面的前端工程師，如 Javascript、Jquery、Bootstrap、React.js、Git、Sass 這些技能，都是會在下班以及假日的時間學習，提升自己的實力。

< / p >

< p > 對於新事物有所好奇，渴望能夠不斷地進步和成長，也在平常付諸行動，期待未來開花結果! < / p >

< h2 > 為什麼想加入貝殼放大？ < / h2 >

< p >

我對募資平台的前端架構優化很有興趣，自從我開始接觸前端技術以後就很喜歡去看一些看起來很特別的網站，並且希望自己可以參與其製作，我相信對於需要群眾集資的產品來說，製作一個能夠吸引使用者的網站是一個不可或缺的部分，所以我想從幕資集眾來去提升使用者體驗與UI是個不錯挑戰。

< / p >

< h2 > 最喜歡你們哪些產品？ < / h2 >

< p >

我很喜歡這一次的徵才網頁，完全展現群眾集資的基本概念，用編輯器當作 Layout 能夠馬上吸引工程師的注意，也成功讓貴公司的職缺很有效的傳遞出去，另外「幾米 x 宜蘭幸福轉運站」這個網站的設計我也很喜歡，網頁配色很繽紛卻不會讓人有眼花撩亂的感覺，有種讓人走進幾米童話的真實感。

< / p >

< h2 > 我能為你們帶來什麼貢獻？ < / h2 >

< p >

以這次的徵才網站來說，我會希望增加一些小功能，像是刪除 Tab 跟新增 Tab 並在網頁內容的最尾端加上一個閃爍的 Bar 另外在每個文字段落前面加上虛線，讓使用者更有在使用編輯器的感覺。

< / p >

< h2 > 目標？夢想？ < / h2 >

< p >

從開始接觸前端技術以後，我就一直期盼自己可以成為一位出色的工程師，但老實說我不知道需要做到什麼樣的地步才能稱得上是出色的工程師，所以我只能不斷的精進自己並把自己推到更高的地方，我相信加入『貝殼放大』會是我完成夢想的第一步也是最重要的一步，在未來的日子裡，我不只會把自己不斷不斷的放大，也會讓周遭的人事物因為我的成長而有所收穫，最後謝謝貝殼放大各位耐心看完我的履歷，希望你們會喜歡，謝謝。

< / p >
